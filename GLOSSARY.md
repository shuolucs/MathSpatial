# Glossary: Chinese Geometric Annotations

Some geometric diagrams in MathSpatial contain Chinese text annotations from original educational materials. This glossary provides English translations.

## View Labels (视图标注)

| Chinese | Pinyin | English |
|---------|--------|---------|
| 正视图 | zhèng shì tú | Front View |
| 主视图 | zhǔ shì tú | Front View (alternative) |
| 侧视图 | cè shì tú | Side View |
| 左视图 | zuǒ shì tú | Left View |
| 右视图 | yòu shì tú | Right View |
| 俯视图 | fǔ shì tú | Top View |
| 三视图 | sān shì tú | Three Views (orthographic projections) |
